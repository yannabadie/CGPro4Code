---
description: Open an interactive ChatGPT 5.5 Pro REPL in the terminal.
---

The user wants to chat directly with GPT-5.5 Pro. Tell them to run:

```bash
cgpro chat
```

in their own terminal — this is an interactive REPL, not something to
spawn from this conversation (you won't be able to relay their inputs
in real time).

Inside the REPL the user can type `:web off`, `:save <name>`,
`:reset`, `:quit`, etc. Multi-line input by ending a line with `\`.
